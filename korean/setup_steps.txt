API 키 발급받기 (무료):

    https://aistudio.google.com 사이트를 엽니다 (필요하다면 구글 계정으로 로그인하세요).
    https://aistudio.google.com/projects 로 이동합니다.
    "Create a new project"를 클릭하여 새 프로젝트를 만듭니다. (기본적으로 무료 등급으로 설정됩니다. 사용량 제한은 있지만 비용은 전혀 들지 않습니다.)
    https://aistudio.google.com/api-keys 로 이동합니다.
    "Create API key"를 클릭하여 키를 생성합니다 (방금 전 단계에서 만든 프로젝트를 선택해서 만드세요).
    키를 생성한 후, 해당 줄의 맨 오른쪽을 보면 API 키를 복사할 수 있는 버튼이 있습니다. ("Set up billing"이라는 파란색 글씨 바로 오른쪽에 있습니다.)

브라우저에서 html 파일(site.html)을 엽니다 (파일을 더블 클릭하면 보통 바로 열립니다). 화면 맨 위에 복사한 API 키를 입력하고, 그 오른쪽에 있는 "Save" (저장) 버튼을 클릭하세요.
이제 generation_steps.txt 파일을 열고 안내에 따라 공부할 내용을 준비해 보세요! :)
