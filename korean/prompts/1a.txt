첨부된 파일들은 대학 강의 슬라이드입니다.
첨부된 문서(강의 내용이 담긴 PDF)를 사용하여 시험에 관련된 모든 내용을 정리해 주세요.
정보의 형식은 AI 튜터가 있는 웹사이트에서 연습할 수 있도록 다음과 같이 조정되어야 합니다:

# AI 튜터 학습 콘텐츠 제작 가이드

## 1. 목표

제공된 문서(예: 질문 목록 및 강의 슬라이드)를 바탕으로 `CONTENT_DATA`라는 단일 JavaScript 변수를 생성하세요. 이 변수는 AI 기반 학습 튜터를 위한 구조화된 지식 베이스 역할을 합니다.

## 2. 구조화의 기본 원칙

### A. 선형적 순서 대신 의미론적 그룹화

가장 중요한 작업은 문서의 내용을 주제별로 연관된 소수의 **학습 카드(Lernkarten)**로 그룹화하는 것입니다. 질문 목록의 각 개별 질문마다 별도의 카드를 만들지 **마세요**. 대신 논리적으로 관련된 주제들을 하나의 상위 카드로 묶으세요.

* **좋은 카드 제목 예시:** "병렬 처리의 기초", "프로세스, 스레드 & 동기화", "군집 지능 알고리즘".
* **나쁜 구조 예시:** "프로세스란 무엇인가?", "스레드란 무엇인가?", "차이점은 무엇인가?"에 대해 각각 카드를 만드는 것. 대신 이들을 "프로세스, 스레드 & 동기화" 카드에 하나로 통합하세요.

### B. 키워드, 설명, 튜터 지식의 명확한 분리

명확하고 상호작용 가능한 학습 인터페이스를 만들기 위해 각 콘텐츠 항목은 다음 원칙에 따라 구성되어야 합니다:

1. **`string` (키워드):** 보이는 텍스트입니다. **매우 간결**해야 하며, 이상적으로는 1~3단어여야 합니다. 개념의 앵커나 라벨 역할을 합니다.
* *좋음:* `"프로세스"`, `"암달의 법칙"`, `"장점"`
* *나쁨:* `"프로세스는 독립된 메모리 영역을 가진 실행 중인 프로그램이다."`


2. **`tooltip` (설명):** 마우스를 올렸을 때 나타나는 "1차 정보"입니다. **짧은 정의, 가장 중요한 공식 또는 간결한 예시 목록**이 여기에 들어갑니다. 툴팁은 유익하되 짧게 유지하세요. `<strong>` 및 `<br>`과 같은 HTML 태그는 허용되며 권장됩니다.
3. **`verbose_tutor_info` (튜터 지식):** **AI에게만 보이는** "2차 정보"입니다. 심도 있는 질문을 던질 수 있도록 포괄적인 설명, 대체 정의, 흔한 오해, 예시, 튜터에 대한 구체적인 지시 사항이 여기에 포함됩니다.

### C. 최우선 목표로서의 학습 카드 가독성

최종 웹페이지에서 `string` 요소의 배치는 논리적이고 읽기 쉬워야 합니다.

* **라벨이 있는 목록:** 한 줄이 어떤 개념과 그 하위 항목들을 나타낼 때, 첫 번째 `string` 요소는 라벨 역할을 해야 하며 **콜론**으로 끝나야 합니다.
* *예시:* `[{ "string": "스레드 조직 형태:" }, { "string": "Dispatcher/Worker" }, ...]`
* *결과:* • 스레드 조직 형태: **Dispatcher/Worker** **Team** **Pipeline**


* **명확하게 명명된 주요 주제:** `"특징:"`과 같은 모호한 라벨 대신, 주요 개념 자체를 라벨로 사용해야 합니다.
* *나쁨:* `스레드 특징:`
* *좋음:* `스레드` (특징은 툴팁이나 `verbose_tutor_info`에 기재)



## 3. 상세 JSON 구조

기본 구조는 원래 지침과 동일하게 유지되지만, 위에 언급된 원칙들에 의해 보완됩니다.

```javascript
const CONTENT_DATA = [
  {
    // 이것은 주제별 학습 카드입니다 (예: "프로세스, 스레드 & 동기화")
    "title": {
      "string": "상위 학습 카드의 제목",
      "tooltip": "카드의 상위 주제에 대한 짧은 설명.",
      "verbose_tutor_info": "이 카드의 일반적인 학습 목표 및 튜터가 집중해야 할 사항에 대한 지침."
    },
    "content": [
      // 내부의 각 배열은 학습 카드의 한 줄입니다.
      [
        // 줄 안의 각 객체는 보이는 요소(키워드)입니다.
        {
          "string": "간결한 키워드 1",
          "tooltip": "키워드 1에 대한 짧은 정의 또는 설명.",
          "verbose_tutor_info": "튜터를 위한 키워드 1에 대한 포괄적인 세부 정보 및 문맥."
        },
        {
          "string": "간결한 키워드 2",
          "tooltip": "키워드 2에 대한 짧은 정의 또는 설명.",
          "verbose_tutor_info": "튜터를 위한 키워드 2에 대한 포괄적인 세부 정보 및 문맥."
        }
      ],
      // 카드의 다음 줄
      [
        {
          "string": "추가 개념",
          "tooltip": "이 개념에 대한 설명.",
          "verbose_tutor_info": "튜터를 위한 추가 지식."
        }
      ]
    ]
  }
];

```
